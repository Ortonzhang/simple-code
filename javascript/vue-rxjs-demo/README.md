# vue-rxjs-demo

页面有倒计时和数据请求，其中load按钮是调用服务器接口，需要手动启动`koa-rxjs-server`项目

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
